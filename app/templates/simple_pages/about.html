About this Project

This project is my assesment for the study program "Software Engeneering Foundations" at CODE University .

My motivation for this Address Changer application came from the struggle you face when you're moving to a new location and you need to change your address for all your accounts (e.g. bank accounts, insurance accounts, internet accounts etc. )

If you are interested in finding out how this site was developed and deployed, visit my Github Repository for this project.
